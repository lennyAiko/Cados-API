## Run Locally

Clone the project

```bash / cmd
  git clone https://github.com/lennyAiko/Cados-API.git
```

Go to the project directory

```bash / cmd
  cd my-project
```

Install modules

```bash / cmd
  pip install -r requirements.txt
```

Start the server

```bash / cmd
  python manage.py runserver
```
